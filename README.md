## 🖼️ What Is anom-artsy?

anom-artsy is the **art asset library** for the entire Anom Originals ecosystem. It contains all visual collectibles, reward icons, mood expressions, and special items used across AO products and platforms.

This is where the visual magic lives — from bling badges to ghost animations, every creative asset is organized, versioned, and ready for export.

---

## 📂 Folder Structure

```
anom-artsy/
├── assets/
│   ├── bling/           # 💎 Premium collectible icons
│   ├── achievements/    # 🏅 Achievement and milestone badges
│   ├── moods/           # 😊 Mood and emotion expressions
│   ├── potions/         # 🧪 Potion and power-up visuals
│   ├── crowns/          # 👑 Crown and rank indicators
│   ├── ghosts/          # 👻 Ghost character assets
│   ├── hearts/          # 💜 Heart and affection icons
│   └── special/         # ✨ Limited edition and seasonal items
├── exports/             # 📤 Production-ready exported assets
├── raw/                 # 🖌️ Source files and working drafts
└── README.md
```

## 🎯 Asset Categories

| Category | Description | Use Case |
|----------|-------------|----------|
| **Bling** | Premium collectible icons | Reward tiers, user profiles |
| **Achievements** | Milestone and accomplishment badges | Gamification, progress tracking |
| **Moods** | Emotion and expression graphics | User status, reactions |
| **Potions** | Power-up and boost visuals | In-app rewards, special abilities |
| **Crowns** | Rank and status indicators | Leaderboards, user levels |
| **Ghosts** | Ghost character variations | Avatars, animations, branding |
| **Hearts** | Affection and appreciation icons | Social features, engagement |
| **Special** | Limited edition and seasonal items | Events, promotions, holidays |

---

## 📤 Workflow

1. **Create** — New assets start in `raw/` as working files
2. **Refine** — Polished versions move to the appropriate `assets/` subfolder
3. **Export** — Production-ready files are placed in `exports/` for use by other repos
